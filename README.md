# Take-a-Break
Small Python program to notify you to take a break, every 45 minutes.

Requires `plyer`, `schedule` and `pyobjus` for macOS

1. Install `plyer`, `schedule` and `pyobjus`
   
   `pip install plyer schedule pyobjus`
   
2. Run the program directly
 
   `python3 main.py`
   
   Or run it with a cronjob.

3. To exit, press 'E'
